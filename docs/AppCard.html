Class: AppCard

AppCard()

A custom HTML element representing a job application card in the View Applications Page. It uses the Shadow DOM for style encapsulation and accepts a `data` object to populate the content. The card displays a company (logo, name, industry), job title, location, pay, date of application with an option to view more details about the company and skill assessment.
