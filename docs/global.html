payScore()

payScore() Given a numeric jobSalary (e.g. 80000) and a user preference salary (the satisfy-point salary), returns a 0–100 score: If prefSalary is undefined or zero => return 50 (neutral) If jobSalary > prefSalary => 100 intolerable point = 0.72 (my personal take) // anything below this score a 0 If (intolerable point < jobSalary < satisfy point) => I use ((jobSalary - intolerable) / (satisfy - intolerable)) * 100
